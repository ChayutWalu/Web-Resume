# Resume Website

A modern, responsive resume website built with HTML, CSS, and JavaScript.

## Features

- ✨ Modern and professional design
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎨 Smooth animations and transitions
- 🚀 Fast loading and optimized
- 🔍 SEO-friendly
- ♿ Accessible

## Getting Started

1. Open `index.html` in your web browser to view the website
2. Edit the content in `index.html` to add your personal information
3. Customize colors and styles in `styles.css`
4. Modify interactions in `script.js` if needed

## Customization Guide

### Personal Information
In `index.html`, replace the following placeholders with your information:
- **Your Name** - Replace all instances with your actual name
- **Job Title** - Update "Full Stack Developer | Software Engineer"
- **Contact Info** - Update email, phone, location
- **Social Links** - Update LinkedIn, GitHub, Twitter URLs
- **Profile Image** - Replace the placeholder image URL

### Work Experience
Update the timeline items in the Experience section with your work history:
- Company names
- Job titles
- Employment dates
- Job descriptions and achievements
- Technologies used

### Education
Update the education cards with your educational background:
- Degree names
- Institution names
- Graduation dates
- Relevant coursework or achievements

### Certificates
Update the certificate cards with your certifications:
- Certification names
- Issuing organization
- Issue and expiry dates
- Brief description
- Certificate verification links

### Skills
Modify the skill categories and individual skills:
- Adjust skill names
- Change proficiency percentages (width values)
- Add or remove skill categories

### Colors
To change the color scheme, edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563eb;    /* Main brand color */
    --secondary-color: #1e40af;   /* Secondary brand color */
    --text-color: #1f2937;        /* Main text color */
    --text-light: #6b7280;        /* Light text color */
    --background: #ffffff;        /* Main background */
    --background-alt: #f9fafb;    /* Alternative background */
}
```

### Favicon
The favicon uses your initials. To change it:
1. Edit the SVG code in the `<link rel="icon">` tag
2. Change "YN" to your initials
3. Optionally change the background color
